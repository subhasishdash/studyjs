Connect to a Webcam in JavaScript and HTML5

Posted on 20th March 2016 at 9:00 PM


We can easily connect to a webcam in Html5 and Javascript. To do so we will need the following things :
  • 1) A browser supporting HTML5
  • 2) A WebCam
  • 3) A Html5 Video tag.
  • 4) getUserMedia

So, lets start building the Webcam streaming by creating a video tag in Html: 
<video id="webcam" autoplay></video>

Now to connect to the webcam we need these JavaScript snippets: 
//Get reference to html video tag:
var video = document.querySelector('#webcam');
//getUserMedia according to current browser.
navigator.getUserMedia = navigator.getUserMedia || navigator.webkitGetUserMedia || navigator.mozGetUserMedia || navigator.msGetUserMedia || navigator.oGetUserMedia;

//If Html5 is supported in your browser ,
//you will get the navigator.getUserMedia otherwise it will be undefined.
if (navigator.getUserMedia) {       
  //prompt the user to give browser level permissions to open the WebCam
  //set video to true({video: true}) if you want only video.
    navigator.getUserMedia({video: true}, streamVideo, videoError);
    //where streamVideo and videoError are callback methods,
    //streamVideo is the method that handles the stream and videoError handles any error 
}

Note:
If you want to stream only video, then set {video: true} like:
navigator.getUserMedia({video: true}, streamVideo, videoError);
otherwise , if you need both audio and video from WebCam, then set {video:true, audio:true} as
navigator.getUserMedia({video: true, audio:true}, streamVideo, videoError);
And now lets write our callback methods 
function streamVideo(streamData){
//now our video tag can only take src data (link to a media),
//Hence, we will need to convert our streamData as
video.src = window.URL.createObjectURL(streamData);
}
function videoError(e) {
	//handle errors
	console.log(e);
}
And thats all , if everything went fine you would be able to connect to your WEBCAM by clicking the below buttons

Both buttons show above will call these two methods : 
//Html code for buttons :
<button class="btn btn-success" onclick="showWebcam()">Show WebCam</button>
<button class="btn btn-success" onclick="showWebcamWithAudio()">Show WebCam With Audio</button>
//show only webcam without audio
function showWebcam () {
	navigator.getUserMedia({video: true}, streamVideo, videoError);
}
//show webcam with audio
function showWebcamWithAudio() {
	navigator.getUserMedia({video: true, audio : true}, streamVideo, videoError);
}
