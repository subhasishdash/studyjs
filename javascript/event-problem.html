How to assign a event to a HTML element in JavaScript?


Let us take a html checkbox element and try to assign a JavaScript event to it. 
<input type="checkbox" id="testbox"/>Sample Checkbox
Sample Checkbox
Great, now let us connect to this above checkbox and add a event handler using it's id in JavaScript 
var btn = document.getElementById('testbox');
btn.onchange = function () {
	alert(this.checked);
}
Now, you can check/uncheck the above checkbox to see the alert box appearing where "true" means it is checked and "false" means it is not checked.

So far so good, let's discuss the problem statement now

As we saw above, the checkbox event works fine without any errors. But what if you want to manage the checkbox from another drop down box without manually checking/unchecking the above created checkbox. Let us see what happens : 
<select id="checkboxmodifier">
<option>Select</option>
<option>Check the above CheckBox</option>
<option>UnCheck the above CheckBox</option>
</select>

Now, let's add event to the above drop down menu to dynamically change the CheckBox. 
var modifier = document.getElementById('checkboxmodifier');
modifier.onchange = function () {
	if (this.value === 'Select') return;
	else if (this.value === 'Check the above CheckBox') btn.checked = true;
	else if (this.value === 'UnCheck the above CheckBox') btn.checked = false;
}
You can select something from the dropdown and check that the checkbox above gets affected but we don't get the events of the checkboxes i.e. a alert message saying "true" or "false" which generally gets called when we check/uncheck manually. So, how to recover that ? Let's fix this.

Solution

To trigger the event on checkbox, we need the call the event programatically which browser calls itself when we operate the checkbox manually. So, we just need to call the event as shown below: 
var modifier = document.getElementById('checkboxmodifier2');
modifier.onchange = function () {
	if (this.value === 'Select') return;
	else if (this.value === 'Check the above CheckBox') btn.checked = true;
	else if (this.value === 'UnCheck the above CheckBox') btn.checked = false;
	btn.onchange();
}
Now, lets change the below dropdown to see if we get the alert event or not. 
<select id="checkboxmodifier2">
<option>Select</option>
<option>Check the above CheckBox</option>
<option>UnCheck the above CheckBox</option>
</select>
Now, you can play with both the drop down menus to understand the concept better. You could also try some examples in a vice-versa manner of what has been done above as a homework.
